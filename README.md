# Criptografia RSA

## Sobre o Projeto
Projeto foi feito com o intuito de resolver o problema do passado pelo Professor de Matemática Discreta:
 Criar um programa que utilizando os conceitos de criptografia RSA e conceitos matemáticos de modulo, criar um programa que:
 - Pessa dois números primos e um coprimo com ele;
 - Pessa uma frase para ser Criptografada e escrita em um arquivo .txt;
 - Pessa os primos e o coprimo para Descriptar a mensageme e escrita em um arquivo .txt.

## Linguagens
- C/C++
  Foi escolhida com o objetivo de utilizar a menor quantidade possivel de outras ferramentas para aprofundar nos conceitos matemáticos 
  e na criação dos algoritimos.
